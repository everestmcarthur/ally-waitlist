Platform Features

Everything you'd expect, and more.

Ally is built from the ground up as a full-featured chat platform with Discord API v10 compatibility — no compromises.

💬

Real-Time Messaging

WebSocket gateway with full opcode support, 18 intent bits, and 50+ event types. Instant delivery, every time.

🏰

Guild System

Servers, channels, roles, permission overwrites, invites, bans, and complete audit logging out of the box.

🔒

48-Flag Permissions

BigInt permission system with role hierarchy and channel overwrites. Fine-grained control over every action.

🎨

Theme Editor

60+ CSS custom properties, 5 built-in presets, and full import/export with .allytheme files. Make it yours.

🧩

Plugin System

Sandboxed iframe plugins with manifest support, gateway hooks, and a built-in plugin store for extensibility.

🤖

Developer Portal

Full application management — bot creation, OAuth2, slash command registration, and webhook integrations.
